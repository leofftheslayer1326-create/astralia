# Astralia

Проект мобильного web app для Astralia by Angelina Pars.

Сейчас внутри лежит статичный кликабельный прототип первой версии. После согласования визуала и логики его можно переносить на Laravel + MySQL + YooKassa.

## Брендбук

- Краткие правила визуала: [`docs/BRANDBOOK.md`](docs/BRANDBOOK.md)
- Слайды брендбука: [`assets/brandbook/`](assets/brandbook/)

## Запуск

```bash
python3 -m http.server 4173
```

Открыть:

```text
http://127.0.0.1:4173
```

## Демо-промокоды

- `ASTROFREE` — делает бесплатными модули "Какие продукты мне подходят" и "Камни-талисманы".
- `STAR15` — скидка 15% на любой модуль.

## Что показывает прототип

- стартовый экран;
- RU/EN переключатель;
- каталог модулей;
- карточку продукта;
- оформление заказа;
- промокоды;
- результат;
- PDF-заглушку;
- личный кабинет;
- партнерский экран;
- визуальный админ-черновик.
