# Astralia Brandbook

Краткая выжимка из брендбука `ASTRALIA by Angelina Pars`. Полные слайды лежат в `/assets/brandbook/`.

## Суть бренда

Astralia — персональный цифровой проводник в мире астрологии, энергии и возможностей.

Характер: премиально, спокойно, точно, персонально. Без давления, дешевой магической атрибутики и агрессивных продаж.

## Палитра

- Deep Space Navy: `#07152E`
- Black Blue: `#0B1020`
- Champagne Gold: `#D8BC84`
- Soft Gold: `#F1E2B8`
- Mystic Violet: `#5D4A86`
- Celestial Blue: `#7DA6D9`

## Типографика

- Заголовки H1/H2: `Cormorant Garamond`, fallback `Georgia`.
- Основной текст, кнопки, цены: `Manrope`, fallback system sans.
- Цены и статистика: табличные цифры, без пляшущей высоты и без отрицательного letter-spacing.

## Логотип

Основной знак — монограмма `A` с орбитой и звездой.

Нельзя использовать религиозные кресты, грубые пиктограммы, мультяшные символы и случайные эзотерические знаки.

## Визуальный Язык

- Темный космос, мягкое шампанское золото, тонкая геометрия.
- Чистые карточки с тонкой золотой линией.
- Мягкое свечение, орбиты, центральная звезда, легкая дымка.
- Иконки в круговых орбитальных рамках.
- Анимация спокойная: мерцание звезд, легкий drift, мягкий hover.

## UI

- Карточки модулей: изображение + название + описание + цена + статус-чип.
- Кнопки: золотая основная, темная вторичная с золотой рамкой.
- Навигация: чистые line-icons, активный пункт подсвечен мягко.
- Переключатель языка: компактный RU/EN без лишних символов поверх текста.
- Промокод, оплата, результат и PDF должны выглядеть частью одной системы.

## Тон Коммуникации

Можно:
- "Открыть ваш персональный разбор"
- "Посмотреть рекомендации"
- "Мягко подсветить следующий шаг"
- "Исследовать энергию периода"
- "Узнать свой потенциал"
- "Выбрать комфортный сценарий"

Нельзя:
- кислотные цвета;
- дешевая магическая атрибутика;
- перегруз символами;
- религиозные кресты;
- мультяшные иконки;
- агрессивные продающие формулировки.
